# Smart Invoice & GST Calculator

Smart Invoice & GST Calculator is a free online billing tool that helps users create GST invoices with automatic subtotal, discount, SGST, CGST, total amount, received amount, balance, and print-ready invoice format.

## Features

- Seller/company details
- Customer billing details
- Shipping details
- Invoice number and date
- Multiple item rows
- Quantity and price calculation
- Unit field
- Item discount
- GST percentage selection
- Automatic SGST and CGST calculation
- Final total amount
- Received amount and balance calculation
- Amount in words
- Print-ready GST bill format
- Responsive design
- Built for Digital Heroes button included

## Tech Stack

- HTML
- CSS
- JavaScript

## Personal Use

I have personally used invoice and GST calculators to check bills, product prices, tax amounts, discounts, and final payable amounts. This tool makes GST invoice calculation faster, simpler, and free.
